OCI-Collection->max

(PHP 5, PECL oci8:1.1-1.2.4)

OCI-Collection->max â€” Returns the maximum number of elements in the collection

Description

OCI-Collection
int max ( void )

Returns the maximum number of elements in the collection.

Return Values

Returns the maximum number as an integer, or FALSE on errors.

If the returned value is 0, then the number of elements is not limited.
