ob_tidyhandler

(PHP 5)

ob_tidyhandler â€” ob_start callback function to repair the buffer

Description

string ob_tidyhandler ( string $input [, int $mode ] )

ob_tidyhandler() is intended to be used as a callback function for ob_start() to repair the buffer.

Example#1 ob_tidyhandler() example

<?php
ob_start('ob_tidyhandler');

echo '<p>test</i>';
?>

The above example will output:

<!DOCTYPE html PUBLIC "-//W3C//DTD HTML 3.2//EN">
<html>
<head>
<title></title>
</head>
<body>
<p>test</p>
</body>
</html>

See also ob_start().
