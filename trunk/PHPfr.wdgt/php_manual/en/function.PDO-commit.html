PDO->commit()

(PHP 5 >= 5.1.0, PECL pdo:0.1-1.0.3)

PDO->commit() â€” Commits a transaction

Description

PDO
bool commit ( void )

Commits a transaction, returning the database connection to autocommit mode until the next call to PDO->beginTransaction() starts a new transaction.

Return Values

Returns TRUE on success or FALSE on failure.

Examples

Example#1 Commit a transaction

<?php
/* Begin a transaction, turning off autocommit */
$dbh->beginTransaction();

/* Change the database schema */
$sth = $dbh->exec("DROP TABLE fruit");

/* Commit the changes */
$dbh->commit();

/* Database connection is now back in autocommit mode */
?>
