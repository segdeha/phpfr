imagesetthickness

(PHP 4 >= 4.0.6, PHP 5)

imagesetthickness â€” Set the thickness for line drawing

Description

bool imagesetthickness ( resource $image , int $thickness )

imagesetthickness() sets the thickness of the lines drawn when drawing rectangles, polygons, ellipses etc. etc. to thickness pixels.

Parameters

image

An image resource, returned by one of the image creation functions, such as imagecreatetruecolor().

thickness

Return Values

Returns TRUE on success or FALSE on failure.

Notes

Note: This function requires GD 2.0.1 or later (2.0.28 or later is recommended).
