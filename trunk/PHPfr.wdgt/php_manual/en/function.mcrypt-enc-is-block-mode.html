(PHP 4 >= 4.0.2, PHP 5)

Description

bool mcrypt_enc_is_block_mode ( resource $td )

This function returns TRUE if the mode outputs blocks of bytes or FALSE if it outputs bytes. (e.g. TRUE for cbc and ecb, and FALSE for cfb and stream).
