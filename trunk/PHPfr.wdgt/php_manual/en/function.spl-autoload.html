spl_autoload

(PHP 5 >= 5.1.2)

spl_autoload â€” Default implementation for __autoload()

Description

void spl_autoload ( string $class_name [, string $file_extensions ] )

This function is intended to be used as a default implementation for __autoload(). If nothing else is specified and autoload_register() is called without any parameters then this functions will be used for any later call to __autoload(). By default it checks all include paths to contain filenames built up by the lowercase class name appended by the filename extensions .inc and .php.
